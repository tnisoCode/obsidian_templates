# Linki 

- zewnętrzny: użyj nawiasów kwadratowych dla tekstu i okrągłych dla adresu URL, (np. do Google: [Google](https://www.google.com))
- wewnętrzny (do pliku, do notatki) [[Person]] lub z aliasem [[Person|Templatka o osobie]].
- dla konkretnego akapitu/ bloku tekstu: [[Person#^69fe5a]]. 

# Format tekstu

- Nagłówki
Użyj znaków # (+ spacja) na początku linii:
# Nagłówek 1
## Nagłówek 2
### Nagłówek 3

- Pogrubienie: **tekst** lub __tekst__ 
- Kursywa: *tekst* lub _tekst_ 
- Połączone: ***pogrubiona kursywa*** (pogrubiona kursywa)
- Zakreślenie: ==tekst==
- Tagi: #tag/nazwa dla wyszukiwania w danym sejfie.

# Listy

Punktowana: Użyj -, * lub + na początku linii:

Jakiś tekst
- Punkt 1
- Punkt 2
  - Podpunkt (wcięcie spacjami lub Tab)

Numerowana: Użyj liczb z kropką:

Jakiś tekst
1. Pierwszy
2. Drugi
3. Trzeci
4. Czwarty
